# WorldCupOS Design System

## Vision
WorldCupOS should feel like a premium real-time global operations platform used during the FIFA World Cup.

The interface should feel cinematic, intelligent, minimal, and alive.

Avoid hackathon aesthetics, emoji-heavy UI, gamer neon overload, and generic SaaS dashboards.

## Design Inspirations
- Linear
- Stripe
- Vercel
- Raycast
- Apple TV sports broadcasts
- Formula 1 telemetry systems

## Visual Style
- Dark luxury interface
- Minimal color palette
- High whitespace
- Soft gradients
- Glassmorphism panels
- Subtle grid overlays
- Animated ambient background

## Colors
Background: #070707
Surface: #111111
Card: rgba(255,255,255,0.04)
Border: rgba(255,255,255,0.08)

Primary Accent: #D4AF37
Secondary Accent: #60A5FA
Danger: #EF4444

Avoid bright green and purple overload.

## Typography
- Large cinematic headings
- Tight font consistency
- Use font weights intentionally
- Minimal ALL CAPS

## Motion
Use Framer Motion subtly:
- fade-in stagger
- smooth hover lift
- sidebar expansion
- animated glow pulses
- scrolling ticker motion

Avoid flashy gamer animations.

## Layout
- More spacing between sections
- Fewer cards visible simultaneously
- Emphasize hierarchy
- Hero-first design

## Sidebar
- Collapsible
- Icon-first
- Animated active states
- Floating appearance
- Premium blur background

## Cards
- Remove thick borders
- Use soft shadows and gradients
- Add hover transitions
- Add subtle glow only on interaction

## Prohibited
- emojis
- rainbow color schemes
- excessive borders
- meme aesthetics
- cluttered UI
